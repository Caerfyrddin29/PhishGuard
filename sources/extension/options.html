Configuration de l’API PhishGuard

Cette page permet de définir l’adresse du backend FastAPI local utilisé par l’extension pour analyser les fichiers email. L’objectif est de garder une configuration claire, simple et sûre pour la démonstration comme pour l’usage quotidien.

Adresse du backend

La version actuelle de l’extension est configurée pour fonctionner avec un backend local sur http://127.0.0.1:8000 ou http://localhost:8000.

Rappel d’utilisation

  • Lance d’abord le backend FastAPI en local.
  • Ouvre ensuite ton webmail et exporte le message au format .eml ou .msg.
  • Dépose ce fichier dans le panneau PhishGuard pour obtenir un score, un verdict et des raisons détaillées.
