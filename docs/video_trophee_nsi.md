# Plan conseillé pour la vidéo Trophée NSI

## Structure générale

Le règlement demande :
- une présentation rapide du projet d'environ 30 secondes ;
- une démonstration détaillée ;
- un total ne dépassant pas 3 minutes.

## Proposition simple

### Partie 1 - présentation rapide
- problème traité : le phishing ;
- objectif : analyser automatiquement un email ;
- résultat attendu : score, verdict et explications.

### Partie 2 - démonstration
- lancer l'API ;
- ouvrir `/docs` ;
- envoyer un fichier `.eml` vers `/analyze/file` ;
- montrer le verdict, le score et les raisons ;
- afficher rapidement la structure du code.

## Conseils

- montrer peu de texte à la fois ;
- zoomer sur les parties utiles ;
- privilégier une vraie démonstration ;
- faire apparaître clairement le pipeline : fichier -> API -> analyse -> résultat.
